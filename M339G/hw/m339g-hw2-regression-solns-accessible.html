In all problems below, we perform a simple linear regression with X as the explanatory variable and Y as the response variable. The model is:

\[ Y = \beta_0 + \beta_1 X + \varepsilon \]

with errors \(\varepsilon\) independent of \(X\), normally distributed with mean 0 and common standard deviation \(\sigma\).

Problem 2.1 (5 × 2 = 10 points)

Determine whether each statement is true or false.

  1. The parameter \(\beta_1\) can be interpreted as the mean increase in the response variable \(Y\) per unit increase in the explanatory variable \(X\).
  2. The parameter \(\beta_0\) is the mean of the response variable \(Y\).
  3. The coefficient of determination \(R^2\) can be interpreted as the proportion of variation in \(Y\) that is explained by the linear model.
  4. The coefficient of determination \(R^2\) is defined as the ratio of the residual sum of squares to the total sum of squares.
  5. \(\sqrt{\text{RSS}\over n-1}\) is the appropriate estimate of the standard deviation of the error \(\sigma\).

Solution:

  • (i) TRUE
  • (ii) FALSE
  • (iii) TRUE
  • (iv) FALSE
  • (v) FALSE

Problem 2.2 (10 points)

For a dataset of \(n=10\) observations \((X, Y)\), you are given:

\[ \bar x = 8,\quad \bar y = 10,\quad \sum_{i=1}^{10} x_i^2 = 4000,\quad \sum_{i=1}^{10} x_i y_i = 5000. \]

Determine the fitted coefficients \(\hat\beta_0\) and \(\hat\beta_1\).

Solution (least squares):

\[ \hat\beta_1 = \frac{\sum x_i y_i - n\bar x\bar y}{\sum x_i^2 - n\bar x^2} = \frac{5000 - 10(8)(10)}{4000 - 10(8)^2} = 1.25. \] \[ \hat\beta_0 = \bar y - \hat\beta_1\bar x = 10 - 1.25(8) = 0. \]

Problem 2.3 (10 points)

For a dataset of \(n=20\) observations \((X, Y)\), you are given:

\[ \sum_{i=1}^{20} x_i = 200,\quad \sum_{i=1}^{20} y_i = 300,\quad \sum_{i=1}^{20} x_i^2 = 3000,\quad \sum_{i=1}^{20} y_i^2 = 4600,\quad \sum_{i=1}^{20} x_i y_i = 3200. \]

Determine \(\hat\beta_0\) and \(\hat\beta_1\).

Solution (least squares):

\[ \hat\beta_1 = \frac{\sum x_i y_i - n\bar x\bar y}{\sum x_i^2 - n\bar x^2} = \frac{3200 - \frac{200\cdot 300}{20}}{3000 - \frac{(200)^2}{20}} = 0.20. \] \[ \hat\beta_0 = \bar y - \hat\beta_1\bar x = \frac{300}{20} - 0.2\left(\frac{200}{20}\right) = 13. \]

Problem 2.4 (10 points)

You are given:

\[ \bar x = 4,\quad \bar y = 3,\quad \sum (x_i - \bar x)^2 = 12,\quad \sum (y_i - \bar y)^2 = 1.25,\quad \sum (x_i - \bar x)(y_i - \bar y) = 3. \]

Determine \(\hat\beta_0\) and \(\hat\beta_1\).

Solution:

\[ \hat\beta_1 = \frac{\sum (x_i - \bar x)(y_i - \bar y)}{\sum (x_i - \bar x)^2} = \frac{3}{12} = 0.25. \] \[ \hat\beta_0 = \bar y - \hat\beta_1\bar x = 3 - 0.25(4) = 2. \]

Problem 2.5 (5 points)

For \(n=25\) observations, you are given:

\[ \bar x = 5,\quad \bar y = 3,\quad \sum x_i^2 = 5000,\quad \sum y_i^2 = 1000,\quad \sum x_i y_i = 450. \]

The residual sum of squares is \(\text{RSS}=300\). Find \(R^2\).

Solution:

Total sum of squares:

\[ \text{TSS} = \sum y_i^2 - n\bar y^2 = 1000 - 25(3)^2 = 775. \]

Then:

\[ R^2 = 1 - \frac{\text{RSS}}{\text{TSS}} = 1 - \frac{300}{775} = 0.6129032. \]

Problem 2.6 (5 points)

Source: An old CAS exam from 1995.

You fit a simple linear regression model with dependent variable values \(y_i = i\) for \(i = 1,\dots,5\). You determine that the estimate of the variance of the error term is \(s^2 = 1\). What is the coefficient of determination?

Solution:

Compute \(\bar y = 3\).

\[ \text{TSS} = \sum_{i=1}^5 (i-3)^2 = 4 + 1 + 0 + 1 + 4 = 10. \]

The residual sum of squares is \(\text{RSS} = (n-2)s^2 = (5-2)(1) = 3\).

\[ R^2 = 1 - \frac{\text{RSS}}{\text{TSS}} = 1 - \frac{3}{10} = 0.7. \]
